# Usage

## Run
Run `bolt` in a directory containing a `bolt.yml`.

```bash
bolt
```

## Controls

| Key | Action |
| --- | --- |
| **Type** | Fuzzy search tasks |
| **Enter** | Execute selected task |
| **↑ / ↓** | Navigate list |
| **Esc / q** | Quit Bolt |

## Behavior
- **Interactive**: Commands run in your full shell (TUI yields control).
- **Interrupt**: Press `Ctrl+C` to stop a running task and return to the menu.
